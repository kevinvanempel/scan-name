# Scan & Name MVP

## Wat zit erin
- React Native + Expo + TypeScript
- Expo Router tabs
- SQLite opslag
- media import
- mock AI herkenning
- automatische titel/map/tags/beschrijving
- zoeken
- mappenoverzicht
- instellingen
- freemium teller
- paywall scherm

## Gebruik van het script
Plaats dit script in een lege map en run:

```bash
python setup_scan_name.py
```

Daarna:

```bash
npm install
npx expo start -c
```
